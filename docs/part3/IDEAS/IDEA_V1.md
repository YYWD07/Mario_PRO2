# Exam Part 3 Ideas - phantom

Mario 每隔几秒自动向面朝方向直线发射一个"分身"（棋盘格方案）。分身碰到敌人时直线运动的消失，然后会分成两个分身，一个向东北或者西北（取决于面朝向）另一个朝东南或者西南，这两个碰到敌人消失或者飞出屏幕消失。当最初直线那个如果没碰到敌人就不分两个，直接飞出屏幕范围，分身消失。当Mario真身碰到敌人的时候，死亡，游戏结束，但是如果死亡瞬间屏幕存有分身，那么就遍历挑选最先在屏幕内的分身变化成真身，原真身和其他的分身都消失，游戏继续，真身继续发射分身等等。如果死亡瞬间屏幕内没有分身，那么就结束了。
